# stock_graph
simple graph for stock API


## 실행하기
myapp/db/Daily_db.js

myapp/db/Intraday_db.js

1. 발급 받은 api key를 apikey 변수에 넣기. 
2. db에 데이터 추가
```c
node db/Daily_db.js
node db/Intraday_db.js
npm start
```
